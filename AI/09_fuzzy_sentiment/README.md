# Fuzzy Inference System for Sentiment Analysis

This project implements a fuzzy inference system for sentiment analysis on hypothetical dataset.

## Description

The system uses fuzzy sets for positive, negative, and neutral sentiments, with rules to classify text sentiment based on a sentiment score.

## Files

- `main.py`: Contains the FuzzySet, FuzzyRule, and FuzzyInferenceSystem classes, and example usage.

## How to Run

1. Ensure Python is installed.
2. Navigate to the project directory: `cd 09_fuzzy_sentiment`
3. Run the script: `python main.py`

## Output

The program will classify sentiment for sample sentiment scores.
