# Neuro-Fuzzy Model for Sentiment Analysis

This project creates a neuro-fuzzy model for sentiment analysis on movie reviews, combining neural networks with fuzzy logic.

## Description

The model trains on labeled data to classify sentiment as positive, negative, or neutral, evaluating accuracy on a test set.

## Files

- `main.py`: Contains the NeuroFuzzySentiment class and example usage.

## How to Run

1. Ensure Python, NumPy, and scikit-learn are installed.
2. Navigate to the project directory: `cd 15_neurofuzzy_sentiment`
3. Run the script: `python main.py`

## Output

The program will train the model and print test accuracy and predictions.
