# Artificial Neural Network for MNIST Digit Classification

This project implements an ANN to classify handwritten digits using the MNIST dataset, trained with backpropagation.

## Description

The network is trained on MNIST and evaluated on the test set, experimenting with different architectures.

## Files

- `main.py`: Contains the ANN implementation using TensorFlow/Keras.
- `requirements.txt`: Lists dependencies.

## Dependencies

- tensorflow
- numpy

## How to Run

1. Install dependencies: `pip install -r requirements.txt`
2. Navigate to the project directory: `cd 13_ann_mnist`
3. Run the script: `python main.py`

## Output

The program will train the model and print test accuracy.
